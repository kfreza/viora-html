Blog Thumbnail Hover Effects

Hover each one. All use only the .cs_post_thumb element — nothing else in the card changes.

1 — Zoom + Gradient Overlay
Subtle slow zoom, dark gradient fades up from the bottom. Safe, classic, editorial.
2 — Zoom + Arrow Reveal

A glossy light streak sweeps across with a faint zoom. Sleek, "premium product" feel.

Card lifts with a colored shadow, brand gradient border lights up. On-brand, eye-catching.
5 — Brand Veil Reveal
